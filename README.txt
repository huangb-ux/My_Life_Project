项目名称：咕咕嘎嘎 - 个人作品集网站
团队成员：胡佳鸣（202313008811，队长）、黄标
完成日期：2026年7月
技术栈：HTML5 + CSS3 + JavaScript ES6+ + Swiper.js

=== 项目描述 ===
这是一个双人作品集展示网站，以全屏垂直滑动方式呈现五个核心页面。
网站采用游戏摄影主题的暗色科技风格，通过双缓冲背景视频、3D旋转木马、
漂浮粒子效果、打字机动画等交互手段，打造沉浸式的浏览体验。

=== 功能模块 ===
1. 首页（第1页）：大字标题"咕咕嘎嘎"，渐变发光文字，打字机副标题，入口按钮
2. 作品选择（第2页）：胡佳鸣/黄标两个分支入口卡片，悬停倾斜3D效果
3. 关于我们（第3页）：双成员卡片展示，包含头像、角色、技能标签
4. 评论区（第4页）：带前端验证的留言表单，昵称/邮箱/留言，支持发布和删除
5. 致谢（第5页）：渐变发光致谢文字
6. 展示页弹窗/独立页面：3D旋转木马浏览影像作品，支持拖拽/滚轮/键盘切换，
   双缓冲背景视频切换，视频与图片面板混排

=== 技术实现 ===
- HTML5 语义化标签：header、nav、main、section、article、footer、figure、figcaption
- CSS3：Flex/Grid布局、动画与过渡、3D透视、backdrop-filter毛玻璃、渐变发光文本
- JavaScript ES6+：Swiper全屏滑动、3D旋转木马实现（手动拖拽+惯性+吸附）、
  双缓冲背景视频切换、粒子系统、卡片3D倾斜效果、打字机动画
- Swiper.js v11：全屏垂直滑动，鼠标滚轮+键盘控制
- 评论区：前端完整验证、动态渲染、XSS防护（escapeHtml）
- 背景视频双缓冲：预加载+交叉淡入淡出，避免切换黑屏
